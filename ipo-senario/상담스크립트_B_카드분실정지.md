# 상담 스크립트 B : 카드 분실 / 정지 처리

> 본 문서는 D-Platform 어드바이저 시뮬레이터 시연을 위한 가상 상담 스크립트입니다.
> 실제 고객 정보와 무관한 창작 데이터입니다.

---

## 상담 개요

| 항목 | 내용 |
|------|------|
| 고객 프로필 | 이미영 (34세, 직장인) |
| 상담 목적 | KB국민카드 분실 신고, 이용 정지, 재발급 문의 |
| 상담 채널 | 전화 (ARS → 카드분실 신고 연결) |
| 상담 유형 | 카드 서비스 > 분실/도난 |
| 상담 시간 | 약 7분 |

---

## 대화 스크립트

**[상담원]** 안녕하세요, KB국민은행 카드고객센터입니다. 어떻게 도와드릴까요?

**[고객]** 카드를 잃어버린 것 같아요. 빨리 정지시켜야 할 것 같아서요.

**[상담원]** 네, 바로 도와드리겠습니다. 고객님 명의로 등록된 카드를 확인해 드릴게요. KB국민 My WE:SH 신용카드, KB국민 대중교통카드 두 장이 있으신데, 분실하신 카드가 어떤 카드인가요?

**[고객]** My WE:SH 카드요, 신용카드요.

**[상담원]** 끝번호 3847로 끝나는 카드 맞으시죠? 바로 사용 정지 처리해 드릴게요.

*(처리 중)*

정지 완료되었습니다. 오후 2시 14분부로 사용 정지 처리되었습니다. 마지막으로 카드를 사용하신 게 언제쯤이었나요?

**[고객]** 오늘 점심에 편의점에서 썼는데 그 이후로 지갑이 없어진 것 같아요. 점심 먹은 식당에도 전화해봤는데 없대요.

**[상담원]** 확인해 드리겠습니다. 마지막 승인 내역이 오후 12시 47분 GS25에서 8,500원 결제이고, 그 이후 추가 이용 내역은 없습니다. 다행히 부정 사용은 없는 것으로 확인됩니다.

**[고객]** 다행이다. 그럼 재발급은 어떻게 해요?

**[상담원]** 재발급은 KB스타뱅킹 앱 또는 영업점 방문으로 신청 가능합니다. 앱으로 하시면 신청 후 영업일 기준 3~5일 내 등록 주소로 배송됩니다. 분실 재발급 수수료는 4,000원인데, 앱에서 신청하시면 수수료 면제 이벤트가 적용될 수 있으니 확인해 보시는 걸 추천드려요.

**[고객]** 앱이 편하겠네요. 근데 나중에 카드가 발견되면 다시 쓸 수 있나요?

**[상담원]** 분실 신고로 정지된 카드는 안전을 위해 재사용이 불가합니다. 재발급 시 새 카드 번호로 발급되고, 기존 카드는 자동 폐기 처리됩니다. 혹시 카드가 발견되시면 가위로 절단 후 폐기하시거나 가까운 영업점에 반납하시면 됩니다.

**[고객]** 알겠어요. 앱에서 재발급 신청하면 되는 거죠?

**[상담원]** 네, KB스타뱅킹 앱 > 카드 > 카드신청/재발급 메뉴에서 신청하시면 됩니다. 배송 주소 변경도 신청 단계에서 하실 수 있어요. 정리해 드리면, My WE:SH 신용카드 분실 정지 완료, 부정 사용 없음 확인, 재발급은 앱에서 신청 가능합니다. 더 궁금하신 사항 있으세요?

**[고객]** 아니요, 빨리 처리해주셔서 감사해요.

**[상담원]** 도움이 되어 다행입니다. 좋은 하루 되세요.

## 상담 메타데이터

| 항목 | 내용 |
|------|------|
| 상담 유형 | 카드 서비스 > 분실/도난 |
| 처리 결과 | 카드 정지 완료 (14:14), 부정사용 없음 확인 |
| 언급 상품 | KB국민 My WE:SH 신용카드 |
| 주요 의도 | 카드 분실 신고, 이용 정지 요청, 재발급 방법 문의, 수수료 문의 |
| 상담 결과 | 정지 처리 완료 / 앱 재발급 안내 |

## KMS 지식 추천 트리거 시점

| 순서 | 트리거 발화 | 추천 지식 |
|------|-----------|---------|
| #1 | "카드를 잃어버린 것 같아요" (즉시) | 카드 분실 신고 처리 절차, 카드 즉시 이용정지 방법, 부정사용 확인 방법 |
| #2 | "재발급은 어떻게 해요" | 카드 재발급 신청 방법 (앱/영업점), 재발급 수수료 정책, 분실 재발급 소요기간 |
| #3 | "나중에 카드가 발견되면" | 분실 신고 후 카드 발견 시 처리 안내, 폐기 방법 및 영업점 반납 절차 |
