## 1. D-Platform / AICC (4분)

### 시연 환경
- 시연 시스템: D-Platform2
- 상담 도메인: 금융상담 (가상 시나리오, 실제 데이터 기반 제작)
- 상담 유형:
  - ① 펀드/투자 상품 추천 문의 (50대 직장인, 퇴직 앞두고 투자 상담)
  - ② 카드 분실/정지 처리 요청

### 1-1. 상담 어드바이저 시연
1. 어드바이저 시뮬레이터에서 가상 상담 파일 불러오기 (파일 업로드 방식)
2. 어드바이저로 상담 데이터 전달 및 처리 시작
3. 실시간 화면에서 확인할 항목:
   - 의도 분석 결과 표시
   - RAG 기반 지식 정보 자동 추천 (상품 정보, 관련 약관 등)
   - 상담 유형 자동 분류 결과
4. 상담 종료 후 → 통화 내역 자동 요약 결과 확인

### 1-2. 인사이트 시연
1. 인사이트 팩토리 화면 진입
2. 개인화 구성 설정 후 보고서 생성 실행
3. 생성된 인사이트 보고서 결과 화면 확인
   - 차트 및 분석 데이터 포함

### 1-3. 자동품질평가 시연
1. 품질평가 기본 화면 설명
2. 동일 상담 데이터를 두 가지 방식으로 평가한 결과 비교
   - 기존 규칙 기반 평가 결과
   - LLM 기반 생성형 평가 결과
3. 두 방식의 차이점 및 LLM 평가의 우수성 강조

### 강조 포인트
- RAG 기반 실시간 지식 추천으로 상담 품질 향상
- LLM 자동평가로 품질관리 비용 절감
- 인사이트 자동 생성으로 관리자 리포팅 효율화

## 4. AI 명함 (3~4분)
> **DQC에서 진행**

### 시연 환경
- 시연 방식: 모바일 화면 미러링 (스크린 캐스트)
- 서비스 상태: 클로즈베타 종료, 정식 서비스 준비 중

### 시연 흐름
1. 모바일 미러링 화면 연결 확인
2. 신규 명함 등록
   - 명함 사진 촬영 또는 이미지 등록
   - 음성 녹음 진행
3. 생성된 AI 명함 확인
   - 음성 합성 결과 확인
   - 영상 합성 결과 확인 (아바타 형태)
4. 학습 데이터 등록
   - 프로필 정보, 주요 이력, 전문 분야 등 입력
5. 등록된 학습 데이터 기반 질의응답 시연
   - 명함 소유자에 대한 질문 입력 → AI가 답변

### 강조 포인트
- 음성/영상 합성 기술(CARO, RVOC, AVTC)의 실제 적용 사례
- 단순 디지털 명함을 넘어 AI 기반 인터랙티브 명함으로 차별화

### 시연 환경
- 시연 주소: https://delma.bot.diquest.com
- 계정: admin / diquest5&^%

### 시연 흐름
1. D-Platform 생성형 기술의 에이전트화 워크플로우 소개
   - D-Platform에서 사용한 주요 AI 기능들(요약, 분류, 평가 등)이 델마 워크플로우 (에이전트화)로 연결된 화면 시연
   - **쇼핑에이전트 워크플로우 필요** (플로우 설계가 필요함.)
   - 각 노드 간 데이터 흐름 설명
2. 자사 솔루션 MCP 노드 연계 화면 확인
   - 목업(Mock) 형태로 구현된 MCP 노드 목록 표시
   - 자사 전체 솔루션이 MCP 노드로 연결 가능함을 시각적으로 표현
3. ModelOps 화면
   - 모델 등록, 버전 관리, 배포 현황 확인
   - 대시보드 및 관리 기능 소개

### 강조 포인트
- 개별 AI 기능을 워크플로우로 자동화하는 에이전트 플랫폼
- 자사 전체 솔루션을 MCP 노드로 통합 관리 가능한 확장성
- ModelOps로 AI 모델의 체계적인 생명주기 관리

## 6. AI 프로페서 (3~4분)
> **DQC에서 진행**
### 시연 환경
- 시연 주소: https://dev-aip.ploonet.com/login
- 계정: hallym / hallym
- 서비스 상태: 개발 완료, 프로젝트 수행 마무리 중

### 시연 흐름
1. 강사 음성 녹음
   - 음성 녹음 화면에서 샘플 음성 입력
2. 음성합성 및 아바타 생성 결과 확인
   - 녹음된 음성 기반으로 합성된 음성 확인
   - 생성된 강사 아바타(영상) 확인
3. 강의 콘텐츠 업로드
   - 주차별 강의노트 파일 업로드
   - 영상 스크립트 파일 업로드
4. 자동 생성된 온라인 강의 결과 확인
   - 아바타가 강의하는 완성된 온라인 강의 영상 재생
5. 사용된 핵심 기술 소개
   - AVTC (음성/영상 합성)
   - MACG (콘텐츠 생성)

### 강조 포인트
- 강사의 음성과 강의 자료만으로 온라인 강의 자동 제작
- 교육 콘텐츠 제작 비용 및 시간 대폭 절감
- 음성합성·영상합성·콘텐츠 생성 기술의 통합 적용
