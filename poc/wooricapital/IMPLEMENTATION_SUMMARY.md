## 📋 프로젝트 개요

**목표**: 회원/비회원 고객을 위한 자동차 추천 시스템 (자연어 검색 + AI 채팅)

## 🔍 주요 기능

### 1️⃣ 자연어 검색 엔진
**파일**: `common.js`, `search_data.js`

#### 5개 사전정의 질의
| ID | 질의 | 키워드 | 추천 차량 |
|----|------|--------|----------|
| result_01 | 연비 좋은 SUV | 연비, 하이브리드, SUV | 쏘렌토, 투싼, NX350h, 싼타페 |
| result_02 | 고성능 세단 | 세단, 고성능, AWD | G80, BMW5, K8, 그랜저 |
| result_03 | 가족 7인승 | 가족, 7인승, SUV | 쏘렌토, 싼타페, 투싼 |
| result_04 | 수입차 추천 | 수입, BMW, 렉서스 | BMW5, NX350h |
| result_05 | 3~4천만원 국산차 | 국산, 저렴, 가성비 | 투싼, 싼타페, 쏘렌토, K8 |

#### 검색 알고리즘
- **입력**: 자연어 (예: "연비 좋은 SUV 가족")
- **토크나이즈**: 공백 기준 분할
- **매칭**: 키워드 교집합 계산 → score 계산
- **결과**: score > 0인 모든 질의 표시 (높은 순서대로)
- **디바운스**: 0.5초 (타이핑 중 불필요한 검색 방지)

#### 화면 전환
| 상태 | 표시 |
|------|------|
| 검색창 비어있음 | 기본 추천 섹션 |
| 입력 → 매칭 성공 | 매칭된 결과셋 (복수) |
| 입력 → 매칭 실패 | "검색 결과 없음" + 추천 질의 칩 |

### 2️⃣ 회원 페이지 (member.html)

#### 고객 프로필
**고객 A**: 김준호 (38세, 금융권 대리)
- 관심: 세단 (주행성능)
- 신용점수: 930 (우수등급)
- 대출한도: 최대 8,500만원
- 추천차량: G80, BMW5, K8, 그랜저

**고객 B**: 이수진 (28세, IT 개발자)
- 관심: SUV (연비 효율)
- 신용점수: 885 (우수등급)
- 대출한도: 최대 6,500만원
- 추천차량: 쏘렌토, 투싼, NX350h, 싼타페

#### UI 요소
1. **상단**: 로고 + 검색창 + 네비게이션
2. **메인**: 배너 + 고객별 추천 차량 그리드
3. **사이드바**:
   - 고객 정보 (프로필, 신용점수)
   - 고객 여정 탭 (검색, 클릭, 찜, 담기, 구매)
   - 하단 메뉴

#### 함수
- `renderMain()` - 메인 콘텐츠 렌더링
- `onSelectA(value)` - A 사용자 선택
- `onSelectB(value)` - B 사용자 선택
- `apply()` - 비교 적용
- `setTab(element, tabName)` - 여정 탭 전환
- `prevUser()` / `nextUser()` - 사용자 전환

### 3️⃣ 비회원 페이지 (non_member.html)

#### AI 시나리오 (5단계)
```
① 성별 선택      → "여성이에요"
② 나이대        → "20대 후반이에요"
③ 직업군        → "마케팅 회사 다니고 있어요"
④ 우선순위      → "연비요!"
⑤ 차종 선택     → "SUV가 좋아요"
    ↓
📊 조건 요약 박스 표시
    ↓
🎯 추천 차량 표시 (카드형)
```

#### 메시지 타입
| 타입 | 설명 |
|------|------|
| agent | 봇 메시지 (왼쪽, 자주색) |
| user | 사용자 메시지 (오른쪽, 파란색) |
| progress | 프로그레스 바 (단계 표시) |
| summary | 조건 요약 박스 |
| rec | 추천 차량 카드 그리드 |
| typing | 타이핑 인디케이터 |

#### 애니메이션
- 각 단계별 delay (500~2200ms)
- 타이핑 인디케이터 (점 3개 바운스)
- 메시지 페이드인 애니메이션
- 채팅 패널 슬라이드인

## 🔧 기술 스택

### 언어
- HTML5 + CSS3
- Vanilla JavaScript (ES5)
- No dependencies

### 핵심 구현
- **자연어 처리**: 키워드 기반 교집합 매칭
- **이미지 관리**: 중앙집중식 CAR_IMGS 상수
- **데이터 구조**: JavaScript 객체 (JSON 호환)
- **비동기 처리**: setTimeout 기반 딜레이
- **DOM 조작**: 직접 조작 (Vue/React 미사용)

    ↓
displaySearchResults() (복수 결과셋 렌더링)

    ↓
getCarFromAllResults() (차량 정보 조회)

    ↓
