# 프로젝트 명세서

## 1. 프로젝트 개요

**프로젝트명**: Wooricapital Auto Recommendation Prototype  
**목적**: 우리금융캐피탈 자동차 상품 추천 시스템 프로토타입  
**범위**: 자동차 상품 추천 화면 + 금융상품 안내  
**기술 스택**: HTML5, CSS3, Vanilla JavaScript (라이브러리 최소화)  
**배포**: 정적 웹 페이지 (서버 불필요)  

---

## 2. 주요 기능 요구사항

### 2.1 사용자 정보 수집
사용자로부터 다음 정보를 수집:

- **[고객A] 김준호**
  1.성별 : 남
  2.나이 : 30대
  3.신용점수 : NICE 930점
  4.보유차량 : 더뉴스포티지(NQ5)
  5.기대기능 : 주행성능
  6.선호 차종 : 세단

- **[고객B] 이수진**
  1.성별 : 여
  2.나이 : 20대
  3.신용점수 : NICE 950점
  4.보유차량 : C-CLASS(W206)
  5.기대기능 : 연비
  6.선호 차종 : SUV

- **[비회원고객C] 최승훈**
  비회원일 경우 인기차량 추천, 에이전트 배너 클릭 후 에이전트와 대화

### 2.2 상품 추천
수집된 정보를 기반으로:
- 사용자 조건에 맞는 자동차 상품 필터링
- 사용자 성향에 따른 순위 매김 (추천 점수)
- 상위 N개의 추천 상품 제시

### 2.3 상품 정보 표시
각 추천 상품마다:
- 이미지, 모델명, 브랜드
- 가격, 연비, 주요 사양
- 주요 특징 (하이라이트)
- 추천 점수 또는 이유

### 2.4 금융상품 안내
상품 선택 후:
- 해당 상품에 맞는 대출 옵션 제시
- 예상 월 납부금 계산 (간단한 금리 계산)
- 다양한 대출 기간 및 이율 제시

### 2.5 인터랙티브 기능
- 실시간 필터링 (폼 입력 시 즉시 업데이트)
- 검색 (모델명, 브랜드로 검색)
- 정렬 (가격, 추천도, 연비 등)
- 비교 기능 (선택한 2~3개 상품 비교)

## 3. 화면 구성

### 3.1 초기화면
- 프로젝트 설명 + 시작 버튼
- 또는 직접 입력 폼으로 진입

### 3.2 사용자 정보 입력 화면
- 멀티 스텝 폼 또는 한 페이지 폼
- 진행률 표시 (스텝 폼인 경우)
- 검증 및 에러 메시지

### 3.3 추천 결과 화면
- 추천 상품 목록 (카드 또는 리스트)
- 필터/정렬 옵션
- 각 상품의 상세 정보
- 상품 선택 → 금융상품 화면으로 이동

### 3.4 금융상품 안내 화면
- 선택한 자동차 상품 요약
- 대출 옵션 비교 (다양한 기간/이율)
- 월 납부금 계산 결과
- 최종 선택/신청 버튼

## 7. 참고사항

- **성능**: 데이터 크기가 작으므로 최적화 불필요
- **호환성**: 최신 브라우저 지원 (IE 미지원)
- **보안**: 클라이언트 사이드 전용이므로 주의사항 없음
- **확장성**: 서버 추가 시 API 호출로 쉽게 확장 가능
