# renewal-page | 이미지 → HTML 프로토타입 생성

스크린샷 이미지를 분석하여 **HTML + CSS**로 빠르게 변환하는 자동화 프로젝트입니다.

---

## 🎯 목표

- 📸 주어진 화면의 이미지 분석
- 🎨 색상, 폰트, 레이아웃 100% 유사하게 HTML로 구현
- ⚡ 빠른 프로토타입 개발
- 🔄 수정사항 반복 적용

---

## 🚀 빠른 시작

### 1단계: 스크린샷 제공
```
📸 변환할 스크린샷 이미지 준비
📝 요구사항 작성 (색상, 폰트, 기능 등)
```

### 2단계: 요청하기
```
사용자: "login-page라는 이름으로 이 이미지를 HTML로 변환해줘"
(이미지 첨부 + 요구사항 제공)
```

### 3단계: HTML 생성
```
✅ 폴더 구조 생성
✅ HTML + CSS 작성
✅ spec.md 작성 (요구사항 기록)
✅ 브라우저 미리보기
```

### 4단계: 수정 반복
```
사용자: "버튼 색을 빨강으로 바꿔줘"
→ 해당 파일 수정 → 재제공
```

## 💡 사용 예시

### 예1: 로그인 페이지
```

📸 이미지: login-page.png
🎨 요구사항:
  - 메인색: #2563EB
```

```
└── spec.md
```

## ❓ 자주 묻는 질문

### Q: 정확하게 픽셀 단위로 일치해야 하나요?
**A:** 아니요. 구조, 색상, 레이아웃이 개념적으로 일치하면 됩니다.

### Q: 어떤 이미지 포맷을 지원하나요?
**A:** PNG, JPG, 고해상도 이미지 권장. 스크린샷은 최대한 선명하게 제공해주세요.

### Q: 외부 라이브러리 (Bootstrap, Tailwind 등) 사용 가능한가요?
**A:** 기본은 Vanilla HTML/CSS입니다. 필요시 요청하면 협의 후 추가 가능합니다.

### Q: 모바일 반응형은 자동으로 포함되나요?
**A:** 네. 기본적으로 반응형 디자인으로 구현됩니다.
