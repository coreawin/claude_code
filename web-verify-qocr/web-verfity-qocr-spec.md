## 메뉴 1. 데이터 > 워크스페이스

### 확인 항목
- [x] 우측 상단에 있는 "워크스페이스 추가" 버튼을 이용해 워크스페이스를 추가한다. 워크스페이스 제목은 '테스트 입력 워크스페이스' 모델 유형은 'Layout Detection'을 선택하고 워크스페이스 추가 버튼을 누른다. 추가후에 목록에서 방금 추가한 워크 스페이스 정보가 표현지는지 확인한다., 

### 기대 결과 / 판정 기준
