# YouTube 인기 영상 조회 프로젝트
이 폴더는 매일 인기 영상 조회하는 프로젝트입니다.
**이 폴더에서는 youtube 스킬을 기본으로 사용합니다.**

## 프로젝트 구조
```
getPopulationDays/
└── index.html   (단일 파일, CSS/JS 인라인 포함 — 서버리스)
```

## 프로젝트 목적
현재 인기있거나 트렌드가 있는 **정보성 콘텐츠**를 찾아 분석하고,
이와 유사한 콘텐츠를 빠르게 생성하여 **트렌드에 맞춘 AI 영상**을 제작·업로드하기 위한 정보 수집 도구.

- 타겟 콘텐츠: 정보전달, 역사, 이야기, 교육, 과학/기술, 뉴스, 엔터테인먼트 등 **AI로 재현 가능한 정보성 영상**
- 핵심 지표: 효율성(구독자 대비 조회수) — 트렌드 선점 가능성을 나타내는 핵심 신호
- 최종 활용: 분석 결과 → AI 영상 스크립트 생성 → 트렌드 영상 빠른 업로드

## 검증 실행 방법
- index.html을 브라우저로 직접 열기
- 상단 API Key 입력란에 YouTube Data API v3 키 입력 후 저장
- 검색어 없이 진입 시 자동으로 1주일 기준 효율성 높은 정보성 영상 조회

## 프로젝트 컨셉
1. 유튜브 API를 활용하여 매일 급상승 또는 인기있는 유튜브를 찾아서 제공한다.
2. 급상승·인기 영상은 유튜브 공식 제공 외에도, **구독자 대비 조회수가 10배 이상** 많은 영상을 주로 추천한다.
3. 서버리스 페이지이므로 HTML 파일만으로 동작하는 프로젝트.

## 효율성 정의 및 정렬
- **효율성 = 조회수 ÷ 구독자수** (소수점 1자리, `OOO.O배` 표시)
- YouTube API에 효율성 정렬 옵션 없음 → 클라이언트에서 계산 후 내림차순 재정렬
- **기본 정렬 기준은 효율성**으로 한다 (API order는 viewCount로 충분한 모수 확보)

| 효율성 | 등급 | 색상 |
|--------|------|------|
| 50배 이상 | 🔥 초급상승 | 빨강 |
| 10~50배 | ⚡ 급상승 | 주황 |
| 3~10배 | ↑ 인기 | 노랑 |
| 3배 미만 | — | 회색 |

## 화면 구성

### 공통 상단 영역
- API Key 입력란 (localStorage 저장)
- 검색어 입력창 + 검색 버튼
- 기본 필터: 정렬 / 기간(오늘·1주·1달·3달·전체) / 길이 / 지역
- 고급 필터(펼치기): 카테고리 / 언어 / 화질 / 안전검색

### 최초 진입 / 검색어 없는 경우
- 자동으로 오늘 날짜 기준 기본 검색 조건으로 조회 실행
- 결과를 효율성 내림차순으로 정렬하여 3단 레이아웃으로 표시

### 3단 레이아웃 구성
```
┌──────────────────────────────────────────────────────┐
│  1단: 전체 결과 중 효율성 TOP 영상 목록              │
│       (효율성 내림차순, 스크롤 / 더보기)             │
├───────────────────┬──────────────────────────────────┤
│  2단: 카테고리 A  │  3단: 추천 채널 패널             │
│  (1단에서 자동선정│  (메인 검색과 독립 동작)         │
│   효율성 TOP)     │  YouTube 알고리즘 기반 채널      │
└───────────────────┴──────────────────────────────────┘
```
- 2단: 1단 결과에서 영상 수 가장 많은 카테고리 자동 선정
- 3단: 메인 검색 결과와 무관하게 독립 동작 (별도 새로고침 버튼)

### 영상 카드 구성 요소 (1단·2단)
- 썸네일 (클릭 시 유튜브 이동)
- 영상 제목
- 채널명 · 업로드 날짜
- 채널 카테고리
- 조회수 / 좋아요 / 댓글수 / 구독자수
- 효율성 뱃지 (강조 표시)
- **[AI 분석] 버튼** → 팝업 오픈 (별도 명세 예정, 하단 참고)

## 3단 주제별 트렌드 클러스터 명세

### 목적
Col1 검색 결과 영상들을 주제별로 자동 분류 → "지금 어떤 주제가 잘 되고 있고, AI로 만들기 쉬운가" 한눈에 파악

> **채널 패널 방식은 폐기됨** — 채널은 AI 영상 제작 워크플로우의 중간 단계일 뿐, 직접 도움이 되지 않음

### 핵심 특징
- **추가 API 없음** — `state.allVideos` 클라이언트 재활용
- 메인 검색 결과가 바뀌면 **자동 갱신**
- 1개 영상이 여러 클러스터에 **중복 포함** 가능

### 주제 클러스터 10개
| 아이콘 | 클러스터 | 감지 키워드 |
|--------|---------|-----------|
| 📜 | 역사/사건 | 역사, 조선, 일제, 전쟁, 사건, 독립, 왕, 시대, 제국, 고려 |
| 🔬 | 과학/기술 | 과학, 우주, AI, 인공지능, 반도체, 기술, 연구, 로봇, 물리 |
| 💰 | 경제/투자 | 주식, 투자, 부동산, 경제, 금리, 환율, 코인, 재테크, 자산 |
| 📰 | 뉴스/시사 | 뉴스, 정치, 대통령, 정부, 국회, 선거, 외교, 이슈, 사회 |

| 🌍 | 세계/문화 | 세계, 나라, 문화, 미국, 중국, 일본, 해외, 외국 |

### 클러스터 카드 구성 요소
```

🟢 쉬움 8편  🟡 보통 4편
─────────────────────────────
1. [썸네일] 제목...    🔥 45.2배
2. [썸네일] 제목...    ⚡ 38.1배
3. [썸네일] 제목...    ⚡ 22.4배
```

### 정렬 토글 (클러스터 간 순서)
| 옵션 | 기준 | 목적 |
|------|------|------|
| **효율순** (기본) | 클러스터 평균 효율 내림차순 | 지금 가장 잘 터지는 주제 |
| **영상수순** | 영상 수 내림차순 | 경쟁이 많은 인기 주제 |
| **쉬움만** | 🟢 쉬움 영상만 필터 후 효율순 | AI 즉시 제작 가능한 주제 |

### AI 제작 난이도 3단계 (영상·클러스터 공통)
카테고리 ID + 제목 패턴 휴리스틱 자동 판정. "어려움" 없음 — 어려운 콘텐츠는 이미 카테고리/키워드 필터로 제외됨.

| 레벨 | 표시 | 판정 기준 |
|------|------|---------|
| **1** | 🟢 쉬움 | 카테고리 25/27/28, 또는 "~란?/하는법/역사/분석/비교/설명/진실" 패턴 |
| **2** | 🟡 보통 | 기본값 |
| **3** | 🟠 주의 | 카테고리 22/23/26, 또는 "브이로그/먹방/여행/체험/직접" 패턴 |

## API 호출 흐름

### 1·2단: 영상 검색 (메인 검색)
```
[검색 버튼 / 필터 변경]
    ↓
① search.list (type=video, maxResults=50) — 검색어 없으면 6개 카테고리 병렬
    → videoId 목록 획득
    ↓
② videos.list (statistics, snippet) — 병렬
③ channels.list (statistics)        — 병렬
    ↓
[클라이언트 품질 필터 → 효율성 계산 → 내림차순 정렬]
    ↓
[1단: 전체 TOP / 2단: 최다 카테고리 TOP 렌더링]
```

### 3단: 주제 클러스터 (API 없음)
```
[메인 검색 완료 후 자동 실행]
    ↓
state.allVideos 순회
    ↓
각 영상 제목 → 10개 주제 패턴 매칭 (클라이언트 JS)
    ↓
클러스터별 집계: 영상수, 평균효율, AI난이도 분포, TOP3 영상
    ↓
정렬 기준 적용 (효율순 / 영상수순 / 쉬움만)
    ↓
[클러스터 카드 렌더링] — 추가 API 호출 없음
```

### 일일 쿼터 예상
| 항목 | 단가 | 최대 횟수/일 | 합계 |
|------|------|------------|------|
| 메인 검색 (1·2단) | ~110 | 검색 횟수 | 가변 |
| 3단 주제 클러스터 | **0** | — | **0** |
| **일 기본 할당량** | | | **10,000** |

## 검색 필터 전체 목록 (YouTube Data API v3 search.list 기준)

### 정렬 (order)
| 값 | 설명 |
|----|------|
| `relevance` | 관련도순 (기본값) |
| `date` | 최신순 |
| `viewCount` | 조회수 높은순 |
| `rating` | 평점 높은순 |
| `title` | 제목 알파벳순 |

### 필터 파라미터
| 구분 | 파라미터 | 선택값 |
|------|---------|--------|
| 기간 | `publishedAfter` / `publishedBefore` | RFC 3339 datetime |
| 영상 길이 | `videoDuration` | `short`(4분↓) / `medium`(4~20분) / `long`(20분↑) / `any` |
| 지역 | `regionCode` | ISO 3166-1 alpha-2 (KR, US, JP …) |
| 언어 | `relevanceLanguage` | ISO 639-1 (ko, en, ja …) |
| 카테고리 | `videoCategoryId` | 숫자 ID (10=음악, 20=게임 …) |
| 화질 | `videoDefinition` | `high` / `standard` / `any` |
| 안전검색 | `safeSearch` | `moderate` / `strict` / `none` |
| 라이브 | `eventType` | `live` / `completed` / `upcoming` |
| 자막 | `videoCaption` | `closedCaption` / `none` / `any` |
